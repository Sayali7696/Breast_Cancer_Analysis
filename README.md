# Breast_Cancer_Analysis
Clinical data analysis of breast cancer dataset using SAS OnDemand for Academics
## Objective
Perform exploratory analysis of breast cancer data using SAS.

## Dataset
Breast Cancer Wisconsin Dataset

## SAS Procedures Used
- PROC IMPORT
- PROC CONTENTS
- PROC PRINT
- PROC FREQ
- PROC MEANS
- PROC SGPLOT
- PROC CORR

## Analysis Performed
- Data import
- Frequency analysis
- Summary statistics
- Histogram
- Boxplot
- Correlation analysis

## Software
SAS OnDemand for Academics
